interface GettableData

Returns the ith value as a ByteBuffer. Note: this method always return the bytes composing the value, even if it is not of type BLOB. That is, this method never throw an InvalidTypeException. However, if the type is not BLOB, it is up to the caller to handle the returned value correctly.
