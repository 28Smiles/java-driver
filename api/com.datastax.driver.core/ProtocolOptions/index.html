ProtocolVersion getProtocolVersionEnum ()

The protocol version used by the Cluster instance.

Returns:
Type Details
ProtocolVersion

the protocol version in use. This might return null if a particular version hasn’t been forced by the user (using say {Cluster.Builder#withProtocolVersion}) and this Cluster instance has not yet connected to any node (but as soon as the Cluster instance is connected, this is guaranteed to return a non-null value). Note that nodes that do not support this protocol version will be ignored.
