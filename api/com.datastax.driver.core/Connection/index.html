CloseFuture closeAsync ()

Closes the connection: no new writes will be accepted after this method has returned. However, a closed connection might still have ongoing queries awaiting for their result. When all these ongoing queries have completed, the underlying channel will be closed; we refer to this final state as “terminated”.

Returns:
Type Details
