LoadBalancingPolicy defaultLoadBalancingPolicy ()

The default load balancing policy.

The default load balancing policy is DCAwareRoundRobinPolicy with token awareness (so new TokenAwarePolicy(new DCAwareRoundRobinPolicy())).

Note that this policy shuffles the replicas when token awareness is used, see TokenAwarePolicy#TokenAwarePolicy(LoadBalancingPolicy,boolean) for an explanation of the tradeoffs.
